void ModMemory::shuffleCards ( void  )

Mélanger les cartes dans le vecteur.

Principe : la vue va associer les cartes de son vecteur avec celles du modèle en prenant leurs indices dans leurs vecteurs respectifs, et le modèle va communiquer les informations des MemoryCards, mélangées donc. Tout ça pour dire que pas besoin de mélanger ou synchroniser la vue et le modèle entre eux.
